## Usage

1. Clone this repo

2. Review outdated dependancies and upgrade if needed

```
yarn outdated
```

for eslint

```
npm info "eslint-config-airbnb@latest" peerDependencies
```

2. Install dependancies

```
yarn dev
```

3. Run app

```
yarn dev
```
