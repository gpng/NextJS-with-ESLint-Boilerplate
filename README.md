## Usage

1. Clone this repo

2. Review outdated dependancies and upgrade if needed

```
yarn outdated
```

for eslint

```
npx install-peerdeps --dev eslint-config-airbnb
```

2. Install dependancies

```
yarn
```

3. Run app

```
yarn dev
```
